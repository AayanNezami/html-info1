What is HTML?

HTML (HyperText Markup Language) is the language used to structure web pages. Think of it as the skeleton that tells a browser what pieces (headings, paragraphs, images, links) go where.

<!doctype html>
<html lang="en">
  <head>...</head>
  <body>...</body>
</html>

Basic structure of an HTML file

Every page usually has the following parts:

  • <!doctype html> â€” tells the browser this is HTML5.
  • <html> â€” root element.
  • <head> â€” metadata like title, styles, character set.
  • <body> â€” visible content on the page.

Headings & paragraphs

Use headings <h1>â€“<h6> for titles and <p> for paragraphs.

<h1>Big title</h1>
<h2>Subheading</h2>
<p>This is a paragraph.</p>

Links (anchors)

Use <a href="URL">link text</a> to jump to other pages or sections.

<a href="https://example.com">Visit example.com</a>

Images

Use <img src="file.jpg" alt="description">. Always add alt for accessibility.

<img src="cat.jpg" alt="A grey cat sleeping">

Lists

Ordered lists (<ol>) and unordered lists (<ul>).

<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>

Forms (user input)

Forms collect user info. Use <form> with inputs like text, email, buttons.

<form>
  <label>Name: <input type="text"></label>
  <button type="submit">Send</button>
</form>

Tables

Tables display tabular data using <table>, <tr>, <td>.

Attributes

Tags can have extra information called attributes, like href, src, alt, class, and id.

Semantic tags

Use semantic elements like <header>, <nav>, <main>, <article>, <footer> to describe meaning â€” this helps accessibility and search engines.

Complete simple example

Here's a minimal working page you can copy into the editor to try:

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>My first page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is my first HTML page.</p>
    <a href="https://example.com">A link</a>
  </body>
</html>
